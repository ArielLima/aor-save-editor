Where are my save files?
%USERPROFILE%\AppData\LocalLow\PersonaeGames\Age of Reforging The Freelands\Save
Quick access: Press Win+R, paste the path above, and hit Enter. Each subfolder is a character — open one, then go into SaveData and pick a save slot. The file you want is sav.dat.
